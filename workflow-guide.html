แก้ไขข้อมูล — เลือกช่องทางตามงาน

LINE แก้จุดเล็ก · บันทึกเร็ว · ถาม AI
Dashboard / profile แก้หลายฟิลด์ · Google Drive/Sheet · ตั้งค่าบัญชี
  • ข้อมูลแยกตาม LINE ID — ไม่ปนกับคนอื่น
  • โครงสร้างเริ่มต้น = ที่ปรึกษาอสังหาริมทรัพย์ (เมทัลชีทหยุดชั่วคราว)

ตั้งค่า LINE OA (ทำครั้งเดียว)

  • ปิด Greeting message (การ์ด Get Started เก่า) — ใช้แค่ carousel จาก webhook
  • เปิด Webhook → ชี้ webhook.php · events: follow + message
  • Add เพื่อน → ได้ carousel 6 การ์ดอย่างเดียว · ปุ่มลงทะเบียนอยู่การ์ด 6

ลิงก์เก่า (redirect อัตโนมัติ)

  • index.php → login.php
  • registor.php → login.php
  • register.php?line_id=... (ไม่ล็อกอิน) → login.php

Add LINE (carousel) → login → ลงทะเบียนครั้งเดียว → Dashboard ↔ คุย LINE ทุกวัน
